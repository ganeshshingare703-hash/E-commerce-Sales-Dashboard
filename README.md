# 🛒 E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview

This Power BI dashboard provides a comprehensive analysis of e-commerce sales performance. It helps businesses monitor revenue, profit, customer behavior, product performance, and payment trends through interactive visualizations.

The dashboard enables stakeholders to make data-driven decisions by identifying top-performing products, profitable regions, and sales trends.

---

## 🎯 Business Objective

This dashboard helps to:

- Check total sales and profit
- Understand customer purchases
- Find top-selling products
- Compare sales in different locations
- Analyze payment methods used by customers
- Support better business decisions.
